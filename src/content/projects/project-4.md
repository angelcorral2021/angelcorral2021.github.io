### Descripción
**Aplicación de Banderas** es una aplicación web sencilla que permite a los usuarios buscar y visualizar banderas de diferentes países. La aplicación cuenta con una interfaz minimalista y fácil de usar, ofreciendo una experiencia rápida para quienes buscan información visual de las banderas del mundo.

### Características
- Búsqueda rápida de banderas por nombre del país.
- Visualización de banderas de alta calidad.
- Diseño ligero y responsivo, optimizado para dispositivos móviles y de escritorio.
- Ideal para estudiantes y curiosos interesados en geografía.
