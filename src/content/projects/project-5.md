### Descripción
**Space API** es una aplicación web diseñada para consumir y mostrar datos de los lanzamientos de SpaceX. Permite a los usuarios explorar información sobre lanzamientos espaciales, misiones, cohetes y otros datos relevantes relacionados con la exploración espacial.

### Características
- Visualización de próximos lanzamientos espaciales y detalles históricos.
- Datos actualizados directamente desde la API oficial de SpaceX.
