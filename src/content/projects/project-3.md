### Descripción
**Calculadora Rect** es una aplicación web interactiva desarrollada con React que permite realizar operaciones matemáticas básicas de forma rápida y eficiente. Ofrece una interfaz intuitiva y responsiva que se adapta tanto a dispositivos móviles como de escritorio, proporcionando una experiencia de usuario fluida.

### Características
- Operaciones matemáticas básicas: suma, resta, multiplicación y división.
- Diseño completamente responsivo.
- Rápida y eficiente, aprovechando el rendimiento de React para actualizaciones instantáneas.
