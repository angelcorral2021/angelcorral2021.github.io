### Descripción
**App Citas Veterinarias** es una aplicación web desarrollada en React para gestionar las citas de una clínica veterinaria de forma eficiente. La aplicación permite registrar a los pacientes (mascotas), agendar, modificar o cancelar citas, y almacenar la información del propietario y la mascota en una interfaz amigable y fácil de usar.

### Características
- Gestión de citas en tiempo real.
- Registro de datos de mascotas y propietarios.
- Editar y eliminar citas existentes.
- Diseño responsivo para dispositivos móviles y de escritorio.
